# <a name="586-32-bit-masm"></a>.586 (32-разрядный компилятор MASM)

Включает сборку непривилегированных инструкций для процессора Pentium. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.586**
