# <a name="586p-32-bit-masm"></a>.586P (32-разрядный компилятор MASM)

Включает сборку всех инструкций (включая привилегированные) для процессора Pentium. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.586P**
