# <a name="compiler-error-c3500"></a>Ошибка компилятора C3500

недопустимый ProgID "ProgID"

С инструкцией указан недопустимый ProgID `#import` . Проверьте реестр Windows, чтобы убедиться, что указан допустимый идентификатор ProgID.
