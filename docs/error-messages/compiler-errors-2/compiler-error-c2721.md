# <a name="compiler-error-c2721"></a>Ошибка компилятора C2721

"описатель": недопустимый спецификатор класса хранения между ключевым словом оператора и типом

Преобразования определяемого пользователем типа применяются ко всем классам хранения, поэтому нельзя указать класс хранения в преобразовании типа.
